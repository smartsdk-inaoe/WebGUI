def default.addCamera ( )

Camera management view.

This view allows the creation and edition of the cameras that will be used in the system. For security, it's required that the user is logged in to view the page.

Parameters
(optional)request.args(0)it's considered the id of the camera, thus the form will contain the data stored for that camera to update it.

When the page its accessed without argumenets, it displays a form to add a camera to the system.
